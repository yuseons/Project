### 📜 DB

- member : 회원 정보 테이블
- contents : 상품 정보 테이블
- review : 상품에 대한 리뷰 테이블
- orders : 회원들의 주문 목록 테이블
- order_detail : 회원들의 주문 상세 테이블
- cart : 장바구니 테이블
